## Programa 1 – Desarrollo y Corrección

## Descripción
Este repositorio contiene el desarrollo del **Programa 1**, el cual incluye el diseño lógico, el procedimiento de solución y la implementación en código Python.  
El proyecto fue realizado como parte de una actividad académica, con el objetivo de aplicar conceptos de programación y estructuración lógica de algoritmos.

---

## Objetivo del Proyecto
- Diseñar un algoritmo que determine si un numero de la sucesion de Fibonacci, si es primo, positivo, negativo o 0.
- Representar la lógica del programa mediante un diagrama de flujo.
- Implementar la solución en Python.

---

## Contenido del Repositorio

- **Diagrama programa 1.drawio.png**

- **correccion programa 1.py**

## Diseño del Programa
El diseño del programa se basa en:
- Uso de diagramas de flujo para estructurar la lógica.
- Desarrollo paso a paso del algoritmo antes de su implementación.
- Corrección del código para mejorar su funcionamiento y claridad.
- Aplicación de buenas prácticas básicas de programación.
